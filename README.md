# Learning Spring Boot

Just a **practice project** to get started with Spring Boot and REST APIs.  

## How to Run
1. Install **Java 21** 
2. Open the project in your IDE (IntelliJ, VS Code, etc.)  
3. Run the `LibraryApplication` main class  
4. Open your browser and visit **http://localhost:8080**
